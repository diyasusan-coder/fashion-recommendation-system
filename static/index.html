Architecture & training

The classifier is a convolutional neural network trained from scratch — no pretrained weights — using stacked convolution + max-pooling blocks to learn edge and texture features, followed by dense layers with dropout for regularization before the final 10-way softmax classification.

This is the piece of the project that demonstrates the fundamentals: writing the training loop, handling class imbalance and overfitting, and evaluating with more than just accuracy — a confusion matrix and misclassification analysis below show exactly where the model struggles (commonly: shirts vs. coats vs. pullovers, which are visually similar even to humans at 28×28 resolution).

About This Project

StyleSense pairs a from-scratch CNN with modern zero-shot vision and language models to turn a single photo into a full styling recommendation.

Pipeline

  • Model fundamentals: CNN trained from scratch on Fashion-MNIST — see Model Insights for the full evaluation.
  • Live garment detection: CLIP zero-shot classification, works on any real color photo with no extra training.
  • Color detection: K-means clustering on the uploaded photo extracts the dominant garment color, with a brightness/saturation check so near-black and near-white items are named correctly.
  • Reasoning: Groq-hosted LLM generates occasion & season-aware pairing suggestions.
  • Visual inspiration: Pexels API pulls real reference photos matching the suggested pairing.
  • Persistence: SQLite-backed Lookbook stores every recommendation across sessions.

Why two models?

Fashion-MNIST is 28×28 grayscale — great for learning and evaluating CNN fundamentals, but it has no color information and can't generalize to real photos. Rather than force it onto live uploads, this project keeps that model as the evaluated, from-scratch piece, and uses CLIP for the live demo where real color photos are the whole point.
